# iPtato
An iptables script that controls system access to and from the network.

## Use it
```ssh
wget -N --no-check-certificate https://raw.githubusercontent.com/Aipblock/iPtato/main/iPtato.sh && chmod +x iPtato.sh && bash iPtato.sh
```

### Run again
```ssh
./iPtato.sh
```
